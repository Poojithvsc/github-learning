Quick Command Reference

git status Check current state
git add . Stage all changes
git commit -m "msg" Save snapshot
git pull --rebase Get team changes
git push Share your work
git switch -c branch Create new branch
git log --oneline View history
git stash Save work temporarily

Learning Tip

Don't try to memorize everything at once. Start with Parts 1-2, practice for a few days, then move on. The Feynman Revision document is great for checking if you truly understand the concepts - if you can explain them simply, you've got it!

Happy Learning! Remember: Git is just a tool to save snapshots and share them.
